Pertemuan 5: Penggunaan Modul dan Paket dalam Python

Tujuan Pembelajaran

Pada akhir pertemuan ini, Anda diharapkan dapat:

  • Memahami konsep modul dan paket dalam Python.
  • Mengimpor modul built-in dan eksternal.
  • Membuat dan mengimpor modul sendiri.
  • Menggunakan paket untuk mengorganisir modul.
  • Memahami dan menggunakan virtual environment.

Materi Pembelajaran

1. Pengantar Modul dan Paket

Apa itu Modul?

  • Modul adalah file yang berisi kode Python seperti fungsi, kelas, atau variabel yang dapat diimpor ke dalam program lain.
  • Modul membantu memisahkan kode ke dalam bagian yang lebih kecil dan dapat digunakan kembali.

Apa itu Paket?

  • Paket adalah koleksi modul yang terorganisir dalam direktori dengan file __init__.py.
  • Paket memungkinkan pengorganisasian modul ke dalam hierarki direktori.

2. Mengimpor Modul Built-in

Mengimpor Modul Built-in:

  • Python menyediakan banyak modul built-in yang dapat langsung digunakan.

  • Contoh:

    import math

print(math.sqrt(16))  # Output: 4.0
print(math.pi)  # Output: 3.141592653589793

Mengimpor Fungsi atau Kelas dari Modul:

  • Anda dapat mengimpor fungsi atau kelas tertentu dari modul.

  • Contoh:

    from math import sqrt, pi

print(sqrt(25))  # Output: 5.0
print(pi)  # Output: 3.141592653589793

3. Membuat dan Mengimpor Modul Sendiri

Membuat Modul:

  • Modul dibuat dengan menyimpan kode Python ke dalam file dengan ekstensi .py.

  • Contoh mymodule.py:

    def greet(name):
    return f"Hello, {name}!"

def farewell(name):
    return f"Goodbye, {name}!"

Mengimpor Modul Sendiri:

  • Modul yang dibuat dapat diimpor ke dalam program lain.

  • Contoh:

    import mymodule

print(mymodule.greet("Alice"))  # Output: Hello, Alice!
print(mymodule.farewell("Bob"))  # Output: Goodbye, Bob!

4. Menggunakan Paket

Membuat Paket:

  • Paket dibuat dengan membuat direktori dan menambahkan file __init__.py di dalamnya.

  • Contoh struktur paket:

    mypackage/
    __init__.py
    module1.py
    module2.py

Mengimpor Modul dari Paket:

  • Modul dalam paket dapat diimpor menggunakan notasi titik.

  • Contoh:

    from mypackage import module1, module2

print(module1.function1())
print(module2.function2())

5. Virtual Environment

Apa itu Virtual Environment?

  • Virtual environment adalah lingkungan terisolasi yang memungkinkan Anda menginstal paket Python tanpa mempengaruhi sistem utama.
  • Virtual environment berguna untuk menjaga ketergantungan proyek tetap terisolasi dan terkontrol.

Membuat Virtual Environment:

  • Virtual environment dapat dibuat menggunakan modul venv.

  • Contoh:

    python -m venv myenv

Mengaktifkan Virtual Environment:

  • Virtual environment harus diaktifkan sebelum digunakan.
  • Contoh:

    • Windows:

      myenv\Scripts\activate

    • MacOS/Linux:

      source myenv/bin/activate

Menginstal Paket dalam Virtual Environment:

  • Setelah virtual environment aktif, paket dapat diinstal menggunakan pip.

  • Contoh:

    pip install requests

Menonaktifkan Virtual Environment:

  • Virtual environment dapat dinonaktifkan dengan perintah deactivate.

  • Contoh:

    deactivate

Aktivitas Pembelajaran

Latihan Praktik:

  1. Buat modul calculator.py dengan fungsi add, subtract, multiply, dan divide. Impor dan gunakan modul ini dalam program utama.

    # calculator.py
def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

def multiply(a, b):
    return a * b

def divide(a, b):
    if b == 0:
        return "Cannot divide by zero"
    return a / b

# main.py
import calculator

print(calculator.add(10, 5))        # Output: 15
print(calculator.subtract(10, 5))   # Output: 5
print(calculator.multiply(10, 5))   # Output: 50
print(calculator.divide(10, 5))     # Output: 2.0

  2. Buat paket shapes dengan modul circle.py dan rectangle.py, masing-masing dengan fungsi untuk menghitung luas dan keliling.

    # shapes/circle.py
import math

def area(radius):
    return math.pi * radius ** 2

def perimeter(radius):
    return 2 * math.pi * radius

# shapes/rectangle.py
def area(width, height):
    return width * height

def perimeter(width, height):
    return 2 * (width + height)

# main.py
from shapes import circle, rectangle

print(circle.area(5))         # Output: 78.53981633974483
print(circle.perimeter(5))    # Output: 31.41592653589793
print(rectangle.area(5, 10))  # Output: 50
print(rectangle.perimeter(5, 10))  # Output: 30

import numpy as np

  2. Buat paket utilities dengan modul file_operations.py yang berisi fungsi untuk membaca dan menulis file. Impor dan gunakan modul ini dalam program utama.

    # utilities/file_operations.py
def read_file(file_path):
    with open(file_path, 'r') as file:
        return file.read()

def write_file(file_path, content):
    with open(file_path, 'w') as file:
        file.write(content)

# main.py
from utilities import file_operations

file_operations.write_file("test.txt", "Hello, world!")
content = file_operations.read_file("test.txt")
print(content)  # Output: Hello, world!
