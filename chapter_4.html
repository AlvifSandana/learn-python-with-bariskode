Pertemuan 4: Pemrograman Berorientasi Objek (OOP) dalam Python

Tujuan Pembelajaran

Pada akhir pertemuan ini, Anda diharapkan dapat:

  • Memahami konsep dasar Pemrograman Berorientasi Objek (OOP).
  • Membuat kelas dan objek dalam Python.
  • Menggunakan atribut dan metode dalam kelas.
  • Memahami dan menggunakan konsep inheritance (pewarisan).
  • Menggunakan encapsulation dan polymorphism.

Materi Pembelajaran

1. Pengenalan Pemrograman Berorientasi Objek (OOP)

Apa itu OOP?

  • OOP adalah paradigma pemrograman yang menggunakan "objek" untuk merepresentasikan data dan metode.
  • Konsep utama dalam OOP: Kelas, Objek, Atribut, Metode, Pewarisan, Enkapsulasi, dan Polimorfisme.

2. Membuat Kelas dan Objek

Kelas dan Objek:

  • Kelas adalah blueprint untuk membuat objek.

  • Objek adalah instance dari kelas.

  • Contoh:

    class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        return f"{self.name} says woof!"

my_dog = Dog("Buddy", 3)
print(my_dog.name)  # Output: Buddy
print(my_dog.bark())  # Output: Buddy says woof!

3. Atribut dan Metode

Atribut:

  • Atribut adalah variabel yang dimiliki oleh objek.

  • Contoh:

    class Car:
    def __init__(self, make, model):
        self.make = make
        self.model = model

my_car = Car("Toyota", "Corolla")
print(my_car.make)  # Output: Toyota

Metode:

  • Metode adalah fungsi yang dimiliki oleh objek.

  • Contoh:

    class Car:
    def __init__(self, make, model):
        self.make = make
        self.model = model

    def description(self):
        return f"{self.make} {self.model}"

my_car = Car("Toyota", "Corolla")
print(my_car.description())  # Output: Toyota Corolla

4. Pewarisan (Inheritance)

Pewarisan:

  • Pewarisan memungkinkan satu kelas mewarisi atribut dan metode dari kelas lain.

  • Contoh:

    class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        pass

class Dog(Animal):
    def speak(self):
        return f"{self.name} says woof!"

my_dog = Dog("Buddy")
print(my_dog.speak())  # Output: Buddy says woof!

5. Enkapsulasi dan Polimorfisme

Enkapsulasi:

  • Enkapsulasi menyembunyikan detail implementasi dari pengguna.

  • Contoh:

    class Person:
    def __init__(self, name, age):
        self.__name = name  # Private attribute
        self.__age = age    # Private attribute

    def get_info(self):
        return f"Name: {self.__name}, Age: {self.__age}"

person = Person("Alice", 30)
print(person.get_info())  # Output: Name: Alice, Age: 30

Polimorfisme:

  • Polimorfisme memungkinkan metode yang sama berperilaku berbeda pada objek yang berbeda.

  • Contoh:

    class Bird:
    def speak(self):
        return "Tweet"

class Cat:
    def speak(self):
        return "Meow"

def animal_sound(animal):
    print(animal.speak())

bird = Bird()
cat = Cat()
animal_sound(bird)  # Output: Tweet
animal_sound(cat)  # Output: Meow

Aktivitas Pembelajaran

Latihan Praktik:

  1. Buat kelas Student dengan atribut name dan grade, serta metode get_grade yang mengembalikan nilai siswa.

    class Student:
    def __init__(self, name, grade):
        self.name = name
        self.grade = grade

    def get_grade(self):
        return self.grade

student = Student("Alice", "A")
print(student.get_grade())  # Output: A

  2. Buat kelas Teacher yang mewarisi dari kelas Person, dengan atribut tambahan subject dan metode teach.

    class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

class Teacher(Person):
    def __init__(self, name, age, subject):
        super().__init__(name, age)
        self.subject = subject

    def teach(self):
        return f"{self.name} is teaching {self.subject}"

teacher = Teacher("Mr. Smith", 40, "Math")
print(teacher.teach())  # Output: Mr. Smith is teaching Math

Tugas Rumah:

  1. Buat kelas BankAccount dengan atribut owner dan balance, serta metode deposit dan withdraw.

    class BankAccount:
    def __init__(self, owner, balance=0):
        self.owner = owner
        self.balance = balance

    def deposit(self, amount):
        self.balance += amount
        return self.balance

    def withdraw(self, amount):
        if amount > self.balance:
            return "Insufficient funds"
        else:
            self.balance -= amount
            return self.balance

account = BankAccount("John", 100)
print(account.deposit(50))  # Output: 150
print(account.withdraw(30))  # Output: 120

  2. Buat kelas Rectangle dan Square yang mewarisi dari kelas Shape, dengan metode untuk menghitung luas dan keliling.

    class Shape:
    def area(self):
        pass

    def perimeter(self):
        pass

class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def area(self):
        return self.width * self.height

    def perimeter(self):
        return 2 * (self.width + self.height)

class Square(Shape):
    def __init__(self, side):
        self.side = side

    def area(self):
        return self.side * self.side

    def perimeter(self):
        return 4 * self.side

rectangle = Rectangle(5, 10)
square = Square(4)
print(rectangle.area())  # Output: 50
print(rectangle.perimeter())  # Output: 30
print(square.area())  # Output: 16
print(square.perimeter())  # Output: 16
