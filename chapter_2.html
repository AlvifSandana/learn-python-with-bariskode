Pertemuan 2: Lanjutan Fundamental Python

Tujuan Pembelajaran

Pada akhir pertemuan ini, Anda diharapkan dapat:

  • Menulis program Python dalam file.
  • Menggunakan list, tuple, set, dan dictionary.
  • Memahami dan mengimplementasikan perulangan.
  • Menggunakan kondisi if, elif, dan else.
  • Memahami dan menggunakan perintah break dan continue.

Materi Pembelajaran

1. Menulis Program dalam File

Menulis dan Menjalankan Program:

  • Program Python dapat ditulis dalam file dengan ekstensi .py.

  • Contoh program sederhana:

    def greet(name):
    return f"Hello, {name}!"

print(greet("World"))

2. List, Tuple, Set, dan Dictionary

List:

  • List adalah koleksi terurut yang dapat diubah.

  • Contoh:

    fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # Output: apple
fruits.append("orange")
print(fruits)  # Output: ['apple', 'banana', 'cherry', 'orange']

Tuple:

  • Tuple adalah koleksi terurut yang tidak dapat diubah.

  • Contoh:

    numbers = (1, 2, 3)
print(numbers[1])  # Output: 2

Set:

  • Set adalah koleksi tidak terurut dan tidak memiliki elemen yang duplikat.

  • Contoh:

    unique_numbers = {1, 2, 3, 4, 4, 4}
print(unique_numbers)  # Output: {1, 2, 3, 4}

Dictionary:

  • Dictionary adalah koleksi pasangan kunci-nilai.

  • Contoh:

    person = {
    "name": "John",
    "age": 30
}
print(person["name"])  # Output: John
person["age"] = 31
print(person)  # Output: {'name': 'John', 'age': 31}

3. Perulangan

For Loop:

  • Digunakan untuk mengulang item dalam koleksi (list, tuple, set, dictionary).

  • Contoh:

    fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

While Loop:

  • Digunakan untuk mengulang selama kondisi tertentu terpenuhi.

  • Contoh:

    count = 0
while count < 5:
    print(count)
    count += 1

4. Pengadilan Kondisi

If, Elif, dan Else:

  • Digunakan untuk membuat keputusan berdasarkan kondisi.

  • Contoh:

    age = 18
if age < 18:
    print("Minor")
elif age == 18:
    print("Just turned adult")
else:
    print("Adult")

5. Break dan Continue

Break:

  • Digunakan untuk menghentikan loop sebelum waktunya.

  • Contoh:

    for i in range(10):
    if i == 5:
        break
    print(i)
# Output: 0 1 2 3 4

Continue:

  • Digunakan untuk melewati iterasi saat ini dan melanjutkan ke iterasi berikutnya.

  • Contoh:

    for i in range(10):
    if i % 2 == 0:
        continue
    print(i)
# Output: 1 3 5 7 9

Aktivitas Pembelajaran

Latihan Praktik:

  1. Buat list berisi nama-nama buah dan tampilkan setiap buah menggunakan perulangan for.
  2. Buat program yang menerima input angka dan menentukan apakah angka tersebut ganjil atau genap menggunakan kondisi if-else.

Tugas Rumah:

  1. Buat dictionary yang berisi data diri Anda (nama, usia, pekerjaan) dan tampilkan setiap informasi menggunakan perulangan for.
  2. Buat program yang menerima input angka dan menghitung faktorial dari angka tersebut menggunakan perulangan while.
