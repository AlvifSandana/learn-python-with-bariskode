Pertemuan 1: Pengenalan dan Dasar-Dasar Python

Tujuan Pembelajaran

Pada akhir pertemuan ini, Anda diharapkan dapat:

  • Memahami apa itu Python dan kelebihannya.
  • Menginstal interpreter Python di komputer Anda.
  • Membuat dan menjalankan program Python sederhana.
  • Menggunakan variabel, operator, dan tipe data dasar dalam Python.
  • Menulis komentar dalam kode Python.

Materi Pembelajaran

1. Pengenalan Python

Apa itu Python? Python adalah bahasa pemrograman tingkat tinggi yang dibuat oleh Guido van Rossum dan dirilis pertama kali pada tahun 1991. Python terkenal karena sintaksnya yang mudah dibaca dan dipelajari, serta dukungan komunitas yang besar.

Kelebihan Python:

  • Sederhana dan mudah dibaca.
  • Memiliki banyak library dan framework yang membantu dalam pengembangan aplikasi.
  • Digunakan dalam berbagai bidang seperti web development, data science, machine learning, dan banyak lagi.

2. Menginstal Interpreter Python

Langkah-langkah Instalasi:

  1. Kunjungi situs resmi Python: Buka python.org.
  2. Unduh installer: Pilih installer yang sesuai dengan sistem operasi Anda (Windows, macOS, atau Linux).
  3. Jalankan installer: Ikuti instruksi instalasi. Pastikan untuk mencentang opsi "Add Python to PATH" saat menginstal di Windows.
  4. Verifikasi instalasi: Buka terminal atau command prompt dan ketik python --version untuk memastikan Python telah terinstal dengan benar.

3. Memulai Pemrograman Python

Menulis dan Menjalankan Script Python:

  1. Buka text editor atau IDE: Anda dapat menggunakan editor teks sederhana seperti Notepad atau IDE seperti VSCode atau PyCharm.

  2. Tulis script sederhana:

    print("Hello, World!")

  3. Simpan file: Simpan file dengan ekstensi .py, misalnya hello.py.

  4. Jalankan script: Buka terminal atau command prompt, navigasikan ke direktori tempat file disimpan, dan jalankan perintah python hello.py.

4. Variabel, Operator, dan Tipe Data Numerik

Variabel: Variabel digunakan untuk menyimpan data. Contoh:

x = 5
y = 3.14
name = "Alice"

Operator: Operator digunakan untuk melakukan operasi pada variabel dan nilai. Contoh:

sum = x + y
product = x * y

Tipe Data Numerik: Python memiliki beberapa tipe data numerik seperti integer (int), float, dan complex. Contoh:

a = 10        # integer
b = 20.5      # float
c = 1 + 2j    # complex

5. Penugasan Variabel, String, dan Komentar

Penugasan Variabel: Penugasan variabel adalah proses memberi nilai pada variabel. Contoh:

message = "Hello, Python!"
age = 25

Manipulasi String: String adalah urutan karakter. Anda dapat menggabungkan string menggunakan operator +. Contoh:

full_name = "Alice" + " " + "Smith"

Komentar: Komentar adalah teks yang tidak dieksekusi oleh interpreter Python dan digunakan untuk menjelaskan kode. Contoh:

# Ini adalah komentar
print("Komentar tidak akan dijalankan oleh interpreter")

Aktivitas Pembelajaran

Latihan Praktik: Buat script Python sederhana untuk menghitung luas persegi panjang:

panjang = 5
lebar = 3
luas = panjang * lebar
print("Luas persegi panjang adalah:", luas)

Tugas Rumah: Tulis script Python yang menghitung luas dan keliling lingkaran dengan radius yang diberikan.

radius = 7
luas = 3.14 * radius * radius
keliling = 2 * 3.14 * radius
print("Luas lingkaran adalah:", luas)
print("Keliling lingkaran adalah:", keliling)
